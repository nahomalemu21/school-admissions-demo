# Abay Horizon Academy — School Admissions Demo

A polished, mobile-first school admissions website created as a sales demo for private schools in Ethiopia.

## What the demo includes

- Admissions-open announcement and strong calls to action
- Mobile-first school homepage
- Program cards for KG, Primary and Secondary
- Campus facilities gallery
- Interactive vertical admissions-video concept
- Campus-visit enquiry form
- Full online application form
- Parent testimonial and FAQ sections
- Click-to-call and mobile quick-action bar
- Responsive navigation and subtle scroll animation
- No framework or build step required

## Run locally

Open `index.html` directly, or run a simple local server:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Deploy free with GitHub Pages

1. Create a public GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and this README.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/root`, then save.

## Before using it for a real school

Replace:

- Fictional school name and logo
- Sample phone number, email and address
- Stock photographs with real school photographs
- Program and admissions information
- Form demo behavior with a real backend

## Connect the forms

The demo forms currently show a success notification without sending data. Production options:

- Google Apps Script → Google Sheets
- Formspree
- Supabase
- Firebase
- A custom API endpoint

## Suggested product package

- Five-page or one-page admissions website
- Online admissions form connected to Google Sheets
- Click-to-call, Telegram/WhatsApp and map integration
- One 45–60 second vertical promotional video
- Domain and GitHub Pages/Vercel deployment
- One month of support
